Options

Les options peuvent être utilisées dans la ligne de commande ou cochées dans le formulaire de synchronisation en mode interactif.

Options de copie

  • /s : Copie les sous-répertoires.
  • /e : Copie les sous-répertoires vides. Sans effet si /s n'est pas spécifié.
  • /r : Ecrase les fichiers en lecture seule.
  • /d : Préserve les fichiers plus récents ou de même date. Seuls les fichiers plus anciens sont écrasés. Si la taille est différente, les fichiers sont copiés quelle que soit la date.
  • /Samba : En complément de /d, autorise une différence d'une seconde entre les dates de fichiers pour ne pas réécrire systématiquement les fichiers non modifiés mais stockés dans des systèmes différents. Particulièrement utile pour les sauvegardes sur NAS Samba.
  • /u : Copie seulement les fichiers existants déjà. Aucun nouveau fichier ne sera créé.
  • /z : Synchronise. Supprime les fichiers de la cible qui n'existent pas dans la source.
  • /a : Copie les droits sur les fichiers (ACL) comprenant les droits d'accès (DACL), les paramètres d'audit, le propriétaire. Les droits sur un fichier ne sont modifiés que si le fichier est modifié.
  • /k : Copie les attributs des fichiers et des dossiers (lecture seule, etc.). Les attributs d'un fichier ne sont modifiés que si le fichier est modifié.
  • /x="Liste" : exclut les fichiers et dossiers correspondant aux masques saisis.
    • Les caractères génériques * et ? sont utilisables. Séparer les masques par ;. Exemples :
      • /x="*\Local Settings\*" : exclut le dossier Local settings.
      • /x="*\temp\*;*.bak;*\~*" : exclut le dossier Temp, les fichiers dont l'extension est .bak, et les fichiers ou dossiers dont le nom commence par ~
    • Pour les utilisateurs avancés, les caractères suivants sont utilisables également :
      • # pour un chiffre (plus restrictif que ?)
      • [A-E] pour tous les caractères entre A et E, majuscules ou minuscules, avec accents éventuellement (les e accentués sont exclus car ils sont classés après E dans l'ordre de tri)
      • [!B-D] pour tous les caractères non compris entre B et D.
      • Exemple : Le fichier C:\aM5b.exe ne sera pas copié avec l'option /x="C:\a[L-P]#[!c-e]

Options de comportement

    • /n : ne demande pas de confirmation avant de créer de nouveaux fichiers.
    • /y : ne demande pas de confirmation avant de supprimer des fichiers.
    • /g : ne s'arrête pas en cas d'erreur.
    • /w : demande une confirmation avant de fermer la fenêtre à la fin de la synchronisation (sans effet en mode console)
    • /q : n'évalue pas la taille des fichiers avant de lancer la copie. Plus rapide, mais sans barre de progression (sans effet en mode console)
    • /l ="Nom de fichier" : enregistre toute l'activité dans le fichier de trace spécifié.

     

Ligne de commande uniquement

  • L'option /go lance la synchronisation en mode console. Les messages sont affichés dans la fenêtre de commande en cours, aucune fenêtre n'est ouverte. Si les options /n et /y sont utilisées, aucune intervention de l'utilisateur n'est nécessaire. Les options /g, /q sont forcées, /w est ignorée.

 

Exemple:

  • Synchro "C:\Tools\My Docs.syn" /go lance la synchronisation sauvée dans "My Docs.syn". Il démarre automatiquement.
  • Synchro /Source=C:\Source Target=C:\Target /s /e /n /z /l=C:\Synchro.log synchronise C:\Target sur C:\Source. Il copie les sous-répertoires, même vides, ne demande pas de confirmation avant de créer les fichiers (mais demande avant de supprimer), et supprime les fichiers excédentaires dans la cible. L'acivité est enregistrée dans le fichier C:\Synchro.log.
