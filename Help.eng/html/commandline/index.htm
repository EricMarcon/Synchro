Running from the command line
Synchro

Syntax

Synchro [File.syn] [Source Target] [/Options] [/go]

  • Fichier.syn is a previously saved synchronization file. It contains all the necessary data. If options are added, they will replace the file values.
  • Source and Target are the folders to synchronize.
  • Options
  • /go launches the synchronization in console mode. Messages are displayed in the active command console. If /n and /y options are used, no user intervention is needed. Synchro returns an ErrorStatus: 0 if the copy succeeded, 1 if there was nothing to do or 2 if it failed.
  • /help displays help in the console.

Example:

  • Synchro "C:\Tools\My Docs.syn" /go runs the synchronization saved in "My Docs.syn". It starts automatically.
  • Synchro /Source=C:\Source /Target=C:\Target /s /e /n /z /l=C:\Synchro.log synchronizes C:\Target on C:\Source. It copies subfolders, even empty, does not prompt before creating files (but prompts before deleting), and deletes extra files in the target. Everything is recorder in the C:\Synchro.log file.
