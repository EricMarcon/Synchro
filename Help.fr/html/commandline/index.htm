Exécution à partir de la ligne de commande
Synchro

Syntaxe

Synchro [Fichier.syn] [Source Cible] [/Options] [/go]

  • Fichier.syn est un fichier de synchronisation créé précédemment. Il contient toutes les informations nécessaires, mais si d'autres options sont saisies, elles remplaceront le contenu du fichier.
  • Source et Cible sont les chemins des répertoires à synchroniser.
  • Options
  • L'option /go lance la synchronisation en mode console. Les messages sont affichés dans la fenêtre de commande en cours, aucune fenêtre n'est ouverte. Si les options /n et /y sont utilisées, aucune intervention de l'utilisateur n'est nécessaire. Synchro retourne le résultat des opérations dans la variable ErrorStatus : 0 en cas de succès, 1 s'il n'y avait rien à copier, 2 si la copie a échoué.
  • l'option /help affiche l'aide en mode console.

Exemple:

  • Synchro "C:\Tools\My Docs.syn" /go lance la synchronisation sauvée dans "My Docs.syn". Il démarre automatiquement.
  • Synchro /Source=C:\Source /Target=C:\Target /s /e /n /z /l=C:\Synchro.log synchronise C:\Target sur C:\Source. Il copie les sous-répertoires, même vides, ne demande pas de confirmation avant de créer les fichiers (mais demande avant de supprimer), et supprime les fichiers excédentaires dans la cible. L'acivité est enregistrée dans le fichier C:\Synchro.log.
