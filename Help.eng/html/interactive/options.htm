Interactive mode
Synchro

Use the Tools /Options menu.

The refresh frequency can be adapted :

  • a very short delay (100 ms) makes display fluid but uses more resources.
  • a longer delay (up to 1 second) may be used if display is not essential.

The most recent-used file list can be chosen between 1 and 10. This option will not be active until the program is restarted.
