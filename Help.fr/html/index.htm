Synchro

Synchro permet la copie d'un dossier dans un autre, avec de nombreuses options similaires à celle de Xcopy, ainsi que la synchronisation (les fichiers supprimés dans le dossier source depuis la dernière copie seront supprimés dans le dossier cible). Le programme peut être exécuté en ligne de commande (dans un script par exemple) ou en mode interactif comme une application Windows.

  • Exécution à partir de la ligne de commande
  • Exécution en mode interactif avec les options du programme
  • Options de copie
