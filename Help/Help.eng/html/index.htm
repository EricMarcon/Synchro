Synchro

Synchro allows copying a folder into another, with a full set of options, similar to those of dos XCopy, including synchronisation (files deleted from the source since the last copy will be deleted in the target). It can be run from the command line (to incude it into scripts) or interactively as a Windows application.

  • Running from the command line
  • Interactive mode with program options
  • Copy Options
